# Introduction
Windows 7, 8 & 10 Arcade Game Screen Saver. 

# Getting Started
Guide:
1.	Rename built ArcadeSaver3.exe file to ArcadeSaver3.scr
2.	Copy the file to your C:\Windows\System32 folder
